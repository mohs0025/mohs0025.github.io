Description list and HTML Tag

h1 to h6 (Heading Tags):
These tags are used to define headings on a web page, with "h1" being the highest level and "h6" being the lowest level
"p" (Paragraph Tag):
This tag is used to define a paragraph on a web page.
"a" (Anchor Tag):
This tag is used to create a hyperlink to another webpage or a specific location within the same page
"img" (Image Tag):
This tag is used to insert an image into an HTML document.
"ul" and "li" (Unordered List and List Item Tags):
These tags are used together to create an unordered (bullet) list of items.
